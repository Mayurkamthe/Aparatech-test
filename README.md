# 🚀 OMVSAB Test Portal

**OMVSAB IT SOLUTION** — *We code your Requirements*

A production-ready online MCQ test and workshop platform.

## Tech Stack
- **Backend:** Node.js, Express.js
- **Frontend:** EJS, Bootstrap 5
- **Database:** MongoDB (Mongoose)
- **Authentication:** Session-based (bcryptjs)
- **Email:** NodeMailer (Gmail SMTP)
- **PDF Reports:** PDFKit
- **Payments:** Razorpay

## Setup
```bash
npm install
cp .env.example .env
# Fill in your .env values
node server.js
```
